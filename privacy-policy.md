1. Données collectées

DriftBot peut collecter :

ID utilisateur, serveur, salon

Nom, tag, avatar

Messages uniquement quand une commande l’exige

Actions liées aux commandes du bot (ex : avertissements, logs)

DriftBot ne collecte pas vos messages privés ni d’informations sensibles.

2. Utilisation des données

Les données servent à :

Exécuter les commandes

Gérer les fonctionnalités

Empêcher les abus

Assurer la stabilité du bot

3. Stockage

Certaines données sont temporaires, d’autres stockées pour le fonctionnement du bot.
Aucune donnée n’est vendue ou partagée.

4. Suppression des données

Vous pouvez demander la suppression de vos données via le serveur support :
https://discord.gg/p5ACHKXEDE

5. Sécurité

Les données sont protégées contre :

Accès non autorisé

Perte

Modification

Fuite

6. Utilisateurs de moins de 13 ans

DriftBot n’est pas destiné aux utilisateurs de moins de 13 ans.

7. Contact
